# Description

Crawler data from e-commerce pages like Epay, Amazon

# Technology

- Backend: Nodejs, Expressjs.
- Framework: Cheerio.
- Frontend: React.
- Database: Mongodb.

# How to use

- Run Test:

```
  npm test
```

- Run Backend:

```
  node app.js
```

- Run Frontend:

```
  cd ./crawler_data_frontend
  npm start
```
